## Overview
This repository contains solutions for FastAPI Internship Assignment – Day 1.  
The assignment focuses on building REST APIs using FastAPI and testing them using browser and Swagger UI.

## Endpoints Implemented
1. **/products** – Display all products with total count  
2. **/products/category/{category_name}** – Filter products by category  
3. **/products/instock** – Show only in-stock products with count  
4. **/store/summary** – Store overview with product statistics and categories  
5. **/products/search/{keyword}** – Search products by name (case-insensitive)  
6. **/products/deals** – Show cheapest and most expensive product (Bonus)
